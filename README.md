# Deutsche Erläuterung

## GWP-Datenmodell 13-BE
Dieses Repository fasst alle Ressourcen zusammen, die für das GWP-Datenmodell 13-BE zu Generellen Wasserversorgungsplanungen (GWP) benötigt werden. Die Ressourcen wurden im Rahmen des Projektes [IPW Informationsplattform Wasser](https://www.ipw.sites.be.ch/de/start.html) erarbeitet und werden hier öffentlich zur Verfügung gestellt. Allgemeine Unterlagen der GWP-Begleitdokumentation des Kantons Bern können auf der [Website des Amts für Wasser und Abfall](https://www.bvd.be.ch/de/start/themen/wasser/wasserversorgung/generelle-wasserversorgungsplanung--gwp-.html) abgerufen werden.

## Verzeichnisstruktur
- `DE` (Deutschsprachige Version):

- `FR` (Version française):

# Explication en français 

## Modèle de données PGA 13-BE
Ce référentiel regroupe toutes les ressources nécessaires au modèle de données PGA 13-BE pour les plans généraux d'approvisionnement en eau (PGA). Les ressources ont été élaborées dans le cadre du projet [PIE Plateforme d'information sur l'eau](https://www.ipw.sites.be.ch/de/start.html) et sont mises à disposition du public ici. Les documents généraux accompagnant la planification générale de l’approvisionnement en eau (PGA) du canton de Berne peuvent être consultés sur le [site de l’Office des eaux et des déchets](https://www.bvd.be.ch/fr/start/themen/wasser/wasserversorgung/generelle-wasserversorgungsplanung--gwp-.html).
